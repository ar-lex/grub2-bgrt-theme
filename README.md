
# BGRT theme for GRUB2

A GRUB2 theme that uses UEFI Boot Graphics Record Table (BGRT) as a GRUB2 background image.

## Installation

To install this theme run
```bash
sudo ./install.sh
```
and follow on-screen instructions.
